TUNE_AUTOMATE

Usage

TUNE_AUTOMATE=_CHAR

Description

This hidden parameter allows multiple options to be automatically invoked upon startup of the monolithic task.

Any options required must be supplied on the command line or the user will be prompted as usual. For example: 

    TUNE_AUTOMATE=1,2,3,EXIT

would run the first 3 processing options and then exit.

If EXIT is not given as an option, ECHOMOP will start the menu task ech_menu when the list of options specified by TUNE_AUTOMATE has been exhausted.
