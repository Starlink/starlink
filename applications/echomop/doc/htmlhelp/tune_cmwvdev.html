TUNE_CMWVDEV

Usage

TUNE_CMWVDEV=_REAL

Description

Specifies the threshold for deciding whether the values generated by the order-specific polynomial and the all-order polynomial are consistent at a particular increment.

TUNE_CMWVDEV is the maximum difference between the two (in pixels) which is accepted as consistent.

The default value is 0.5 pixels.
