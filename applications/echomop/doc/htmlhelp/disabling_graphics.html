Disabling Graphics

ECHOMOP supports SOFT=NONE to disable all hard and soft graphics facilities which can save CPU time when such facilities are not required. The SOFT parameter should be set using 

     SOFT=NONE or SOFT=NULL

to disable ECHOMOP graphics.
