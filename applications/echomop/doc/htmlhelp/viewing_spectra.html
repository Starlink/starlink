Viewing Spectra

It is often useful when reducing 'difficult' data, to be able to examine various intermediate 'spectra' generated during the reduction process. ECHOMOP provides a very flexible method of examining any intermediate arrays produced by it or other tasks.

The ech_plot task/option provides zooming, re-binning, re-scaling etc. on any array in the reduction structure. Arrays may be specified by name or by using a the reference names by which ECHOMOP tasks access them.

To obtain a brief directory of reference names use the D(irectory) option in the ech_plot option/task.

To obtain a full directory use the FD option.
