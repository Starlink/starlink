TUNE_CRINTER

Usage

TUNE_CRINTER=_LOGICAL

Description

Determines how cosmic-ray-contaminated pixels are to be treated. (Only used when cosmic-ray checking is enabled by TUNE_CRTRC=TRUE.)

When set to TRUE cosmic-ray pixels are replaced in the output image by interpolated values.

When set to FALSE cosmic-ray pixels are marked with a flag value.

The default is FALSE.
