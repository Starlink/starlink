TUNE_AAACODE

Usage

TUNE_AAACODE=_INTEGER

Description

Used only when the program is making use of the Astronomy & Astrophysics Abstracts system of information categorisation (TUNE_USEAAA=YES).

This system consists of a set of numbers corresponding to topics in Astronomy. TUNE_AAACODE is sused to supply an AAA standard information category for the data being reduced. These parameters are only used in conjunction with the archiving facility, when TUNE_ARCHIVE=YES.

INITIAL SUGGESTED VALUE: 1.
