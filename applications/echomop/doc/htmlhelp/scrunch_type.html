SCRUNCH_TYPE

Usage

SCRUNCH_TYPE=_CHAR

Description

The type of spectrum to be scrunched.

This may be one of the following: 

  - OBJ, the object.
  - STAR, the reference star.
  - ARC, the arc lamp.
