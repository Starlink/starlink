Demo

Summary

ECHOMOP provides a set of demonstration data frames and can run through the main reduction steps using these frames. The demo is constructed of a series of descriptions followed by actual examples of the task usage. To view the demo type 

      % ech_demo -graphics-device

at the command line where -graphics-device- is any legal GKS graphics display name (e.g., "xw").

The text printed during the demo is available in the file $ECHOMOP_DEMO/ECH_DEMO.TXT if you would prefer to read from a printed copy.
