TUNE_DSGMTHR

Usage

TUNE_DSGMTHR=_REAL

Description

The sigma-clipping threshold for deciding which pixels to clip at each iteration of CDF matching.

Normally clipping should be set to 2 to 3 sigma. At each iteration the deviation of each pixel from its predicted value is evaluated in terms of the mean and sigma for all pixels (on a per order basis).
