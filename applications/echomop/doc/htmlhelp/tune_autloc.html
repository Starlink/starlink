TUNE_AUTLOC

Usage

TUNE_AUTLOC=_LOGICAL

Description

Set TRUE if the positions of the orders are to be determined automatically.

When FALSE, the program allows interactive selection of the order centres on a graphic display. This is useful when the user wishes to select a subset of the available orders in the frame, or to review the order centres found by the program.

The default value is FALSE.
