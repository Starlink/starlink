TUNE_SCRADD

Usage

TUNE_SCRADD=_INTEGER

Description

Used only when TUNE_SCRMODE=0, and indicates the number of input bins to be added into each output bin. This is used mainly to increase the signal-to-noise of a weak spectrum, at the expense of resolution.
