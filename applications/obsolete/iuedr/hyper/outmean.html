OUTMEAN

Summary

Output mean spectrum from DATASET to a DIPSO "SP" format file. 

   DATASET         Dataset name.
   OUTFILE         Name of output file.
   SPECTYPE        "SPECTRUM" file type (0, 1 or 2).

Purpose

This command outputs the mean spectrum associated with DATASET to a file that can be read into DIPSO (see SUN/50).

This file is created with type specified by the SPECTYPE parameter (see SUN/50 for SP options), and its name, if not specified, is constructed: 

      <CAMERA><IMAGE>M.sdf    for SP0
      <CAMERA><IMAGE>M.DAT    for SP1 or SP2

In DIPSO SP format, bad points are indicated by having zero intensities. In determining which points in the output file are to be marked "bad", the user-defined data quality bit is used. Since this bit can be arbitrarily edited, faulty data values can be written to the output file, without subsequent information being retained.
